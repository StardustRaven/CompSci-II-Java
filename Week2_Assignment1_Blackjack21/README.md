# java-card-game
Java Blackjack 21

A simple command line Blackjack (21) game written in Java

## How to run
compile and run 'Blackjack21.java'.

The game supports hit / stand gameplay and gracefully exits if the deck runs out of cards

## Files
- Blackjack21.java - main game logic
- Card.java - card representation
- DeckOfCards.java - deck creation and shuffling
